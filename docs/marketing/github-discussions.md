## Frequently Asked Questions

### Q: Is this project officially from Anthropic?
**A:** No. This is an independent open-source project with no affiliation with Anthropic or Claude.ai.

### Q: Will my data be uploaded to the cloud?
**A:** No. All data is stored on your local file system.

### Q: Do I need a Python/Node.js environment?
**A:** No. You only need to install Claude Code CLI.

### Q: Can it be used for commercial purposes?
**A:** Yes, the project uses the MIT license.

## Community Guidelines

1. **Respect others** - Stay friendly and constructive
2. **Stay on topic** - Keep discussions relevant to the project
3. **Search first** - Search for similar discussions before posting
4. **Privacy awareness** - Do not share real medical data in discussions
